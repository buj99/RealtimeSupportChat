1. Abstract

Realtime SupportChat este o extensie ce permite introducerea unei ferestre de chat in site-ul propriu.

2. Introducere

Realtime SupportChat este o aplicatie web ce ofera un script cu ajutorul caruia se poate introduce o fereastra de chat intr-un site. Conversatiile sunt salvate si afisate in meniul dedicat administratorului, unde acesta poate sa si downloadeze scriptul si sa customizeze designul ferestrei de chat.

Scopul aplicatiei:

  • se incadreaza in domeniul business-management
  • aduce o modalitate rapida si eficienta de comunicare cu echipa de suport tehnic
  • usureaza comunicarea client-moderator

3. Structura

Site-ul este de tipul single-page aplication, are un API la care se fac call-uri si un server care retine continutul static al aplicatiei, avand de asemenea o baza de date ce stocheaza mesajele primite de administrator si detalii despre acestea.

3.1 Tehnologii utilizate

  • HTML
  • CSS
  • JS

3.2 Pagina html

Documentul este codat in UTF-8 si este transmis cu un tip media text/html.

Head-ul trebuie sa contina <meta charset="utf-8"> si un titlu.

3.3 Pagina de login

Pagina de login asigura o modalitate utilizatorului de a se loga pentru ca acesta sa poata beneficia de serviciile site-ului.

3.4 Pagina de register

Pagina de register are functionalitatea de a crea conturi noi pentru utilizatorii site-ului.

3.5 Pagina userului (UserMenu)

Aceasta pagina contine toate conversatiile administratorului si o bara de navigatie care permitere iesirea din cont, download-ul scriptului pentru chat si intrarea pe o pagina de costumizare.

3.6 Pagina de costumizare

Pagina de costumizare permite utilizatorului schimbarea temei chat-ului, a culorii scrisului, a dimensiunii fontului si a mesajului de intampinare by default al chat-ului.

3.7 Pagina de descarcare

Pune la dispozitie scriptul pentru chat-ul costumizat de client.

4. Caracteristicile sistemului

Pagina dedicata administratorului contine o coloana cu toate conversatiile ce au fost initiate de catre vizitatorii site-ului. In dreapta coloanei se afla conversatia curenta ce contine in partea de jos o zona de input in care se pot scrie mesaje si un buton pentru trimitre mesaj. Fereastra de chat ce apare pe pagina site-ului contine initial un mesaj de intampinare ce poate fi customizat de catre administrator si o zona de input in care se pot scrie mesaje si un buton pentru trimitre mesaj Pagina de inregistrare contine un camp pentru email, 2 campuri pentru parola respectiv pentru confiramrea parolei si un buton de inregistrare. Pagina de login contine 2 campuri de text pentru emai respectiv parola si un buton de logare. Pagina de customizare a chatului contine un dropbox pentru selectarea temei, unul pentru selectarea culorii textului si 3 radiobuttons pentru alegerea mesajului de intampinare din fereastra de chat, precum si un camp text in care se poate introduce un alt text decat variantele sugerate.

5. Constrangeri

Nu exista constrangeri de utilizare, deoarece site-ul e conceput pentru orice tip de ecran, functionand si pe modul orizontal al unui telefon , tablete, etc.

6. Functionalitatea site-ului

Site-ul este construit in asa fel incat utilizatorii, detinatori de site-uri care administreaza o afacere , fara limita de varsta, sa fie atrasi de interfata placuta si usor de utilizat pentru o comunicare fructuoasa. Utilizatorii pot apartine unei singure categorii, cei logati, pentru a putea beneficia de servicii. Site-ul pune la dispozitie utilizatorului diverse functii precum crearea unui cont, logarea, folosirea contului pentru a comunica cu clientii si customizarea chat-ului cu chestii precum alegerea unui mesaj de intampinare, dimensiunea fontului, culoarea textului, si schimbarea temei de fundal. Chatul nu poate fi pus la dispozitie unui utilizator nelogat.
