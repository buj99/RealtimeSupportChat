Abstract

Este o extensie ce permite introducerea unei ferestre de chat in site-ul propriu

1.Introducere

1.1 Nota document

Realtime Suport Chat este o aplicatie web ce ofera un script cu ajutorul
caruia se poate introduce o fereastra de chat intr-un site. Conversatiile
sunt salvate si afisate in meniul dedicat administratorului, unde acesta
poate sa si downloadeze scriptul si sa customizeze designul ferestrei de
chat.>

1.2 Scopul aplicatiei

Site-ul se incadreaza in domeniul business-management, el furnizeaza un
chat drept modalitate eficienta si rapida de comunicare pentru echipa de
suport tehnic al unui site de business , asadar scopul site-ului este de a
usura comunicarea client-moderator.

2.Descriere de ansamblu

2.1 Functionalitatea site-ului

Site-ul pune la dispozitie utilizatorului diverse functii precum crearea
unui cont, logarea, folosirea contului pentru a comunica cu clientii si
customizarea chat-ului cu chestii precum alegerea unui mesaj de
intampinare, dimensiunea fontului, culoarea textului, si schimbarea temei
de fundal. Chatul nu poate fi pus la dispozitie unui utilizator nelogat.

2.2Clase de utilizatori si caracteristici

Site-ul este construit in asa fel incat utilizatorii, detinatori de
site-uri care administreaza o afacere , fara limita de varsta, sa fie
atrasi de interfata placuta si usor de utilizat pentru o comunicare
fructuoasa. Utilizatorii pot apartine unei singure categorii, cei logati,
pentru a putea beneficia de servicii.

2.3 Mediul de operare

Site-ul este conceput pentru a putea fi accesat de pe orice dispozitiv
care dispune de un browser si de o conexiune la internet.

2.4 Constrangeri de proiectare si implementare

Nu exista constrangeri de utilizare, deoarece site-ul e conceput pentru
orice tip de ecran, functionand si pe modul orizontal al unui telefon ,
tablete, etc.

2.5 Dependente

Site-ul se bazeaza pe o interactiune server-client, in care clientul,
adminul, se poate inregistra sau loga pentru a-si crea un chat
personalizat pe care il poate descarca si utiliza in propriul lui site .

3.Descriere interfata

3.1Design

Fereastra de chat apare in pagina site-ului deasupra celorlalte elemente,
continand initial mesajul de intampinare. Pagina de administrator contine
un buton ce adauga peste continut un meniu cu actiuni suplimentare
(logout, downloadare script).

3.2 Tehnologii utilizate

HTML
CSS
JS

3.3Caracteristicile sistemului

Pagina dedicata administratorului contine o coloana cu toate conversatiile
ce au fost initiate de catre vizitatorii site-ului. In dreapta coloanei se
afla conversatia curenta ce contine in partea de jos o zona de input in
care se pot scrie mesaje si un buton pentru trimitre mesaj. Fereastra de
chat ce apare pe pagina site-ului contine initial un mesaj de intampinare
ce poate fi customizat de catre administrator si o zona de input in care
se pot scrie mesaje si un buton pentru trimitre mesaj Pagina de
inregistrare contine un camp pentru email, 2 campuri pentru parola
respectiv pentru confiramrea parolei si un buton de inregistrare. Pagina
de login contine 2 campuri de text pentru emai respectiv parola si un
buton de logare. Pagina de customizare a chatului contine un dropbox
pentru selectarea temei, unul pentru selectarea culorii textului si 3
radiobuttons pentru alegerea mesajului de intampinare din fereastra de
chat, precum si un camp text in care se poate introduce un alt text decat
variamntele sugerate.
